# Air Quality Regression

Supervised machine learning case study focused on predicting air pollutant concentrations from chemical sensor measurements.

## Objective

The project applies regression techniques to an air quality dataset collected by chemical sensors in an urban environment.

The objective is to analyse the data, prepare the variables and compare different regression approaches for predicting pollutant concentrations.

## Methodology

* Exploratory data analysis
* Missing-value analysis and data cleaning
* Feature preprocessing
* Simple linear regression
* Multiple linear regression
* Decision tree regression
* Train and test evaluation
* Model comparison
* Overfitting analysis

## Models

* Simple Linear Regression
* Multiple Linear Regression
* Decision Tree Regressor

## Evaluation Metrics

* Mean Squared Error
* Coefficient of Determination ((R^2))

## Dataset

Air Quality Dataset from the UCI Machine Learning Repository.

The dataset contains hourly measurements from chemical sensors recorded in an Italian urban area between March 2004 and February 2005.

## Technologies
