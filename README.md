# Machine Learning Case Studies

A collection of supervised machine learning projects developed during my MSc in Artificial Intelligence.

The repository includes practical applications of regression, classification, model evaluation, hyperparameter optimisation and imbalanced dataset handling.

## Case Studies

### [Air Quality Regression](./01-air-quality-regression/)

Regression analysis using chemical sensor measurements from an urban air quality dataset.

Main topics:

* Exploratory data analysis
* Missing-value analysis and data cleaning
* Simple linear regression
* Multiple linear regression
* Decision tree regression
* Model evaluation
* Overfitting analysis

### [Covertype Classification](./02-covertype-classification/)

Classification of forest cover types using cartographic and environmental variables.

Main topics:

* Exploratory data analysis
* Continuous and categorical feature analysis
* Class imbalance analysis
* Feature scaling
* Support Vector Machines
* Random Forest classification
* Cross-validation
* Undersampling and oversampling
* Confusion matrices and classification reports
* Model comparison

### [Hate Speech Classification](./03-hate-speech-classification/)

Classification of text messages as hate speech or non-hate speech using engineered linguistic and sentiment features.

Main topics:

* Exploratory data analysis
* Class distribution analysis
* Correlation and feature importance analysis
* Feature subset comparison
* Random Forest
* Support Vector Machines
* Decision Tree classification
* Hyperparameter optimisation with GridSearchCV
* Accuracy, precision, recall and F1-score
* Final model selection

## Technologies
