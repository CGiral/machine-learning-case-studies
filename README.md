# Machine Learning Case Studies

A collection of supervised machine learning projects developed during my MSc in Artificial Intelligence.

The repository includes practical applications of regression, classification, model evaluation, hyperparameter tuning and imbalanced dataset handling.

## Case Studies

### 1. Air Quality Regression

Regression analysis using air quality sensor data.

Main topics:

- Exploratory data analysis
- Data cleaning and preprocessing
- Simple linear regression
- Multiple linear regression
- Decision tree regression
- Model evaluation
- Overfitting analysis
