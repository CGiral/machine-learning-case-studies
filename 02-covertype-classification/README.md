# Covertype Classification

Supervised machine learning case study focused on classifying forest cover types from cartographic and environmental variables.

## Objective

The project compares Support Vector Machines and Random Forest models using the Covertype dataset.

The analysis also examines the impact of class imbalance and evaluates different sampling strategies.

## Methodology

* Exploratory data analysis
* Analysis of continuous and categorical variables
* Class distribution analysis
* Feature scaling
* Support Vector Machine classification
* Random Forest classification
* Cross-validation
* Confusion matrices
* Random undersampling
* Random oversampling
* Model comparison
* Review of a scientific classification application

## Models

* Support Vector Machine
* Random Forest Classifier

## Evaluation

* Accuracy
* Classification report
* Confusion matrix
* Cross-validation scores

## Dataset

The Covertype dataset contains cartographic variables used to predict seven different forest cover types.
